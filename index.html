Intro to Web Development

What is a Front-end Developer?

Front-end developers are responsible for the parts of a website that users see and interact with.

Front-end Stack

  • HTML
  • CSS
  • JavaScript (TypeSpricpt)
  • ReactJS
  • Bootstrap
  • Foundation
  • Backbone
  • jQuery
  • LESS
  • AJAX

Front-end Responsibilities

Front-end dev is responsible for the interior design of the foundations built by a back-end developer.

They work closely with the designers or UX analysts to bring mockups and wireframes from development to delivery.

What is a back-end developer?

Builds and maintains the technology that powers the server, applicaiton, and database.

Back-end Stack

Application Building:

  • PHP
  • Python
  • Ruby
  • Java
  • .NET

Data Management:

  • MySQL
  • PostgreSQL
  • MongoDB
  • Redis
  • Memcached

What is a full stack developer?

Full stack developers are familiar with each layer of development and can decide how to best implement a certain feature.
